## プロンプト①：キーワード戦略（Agent19.5 Keyword Strategy Agent用）

```
以下の店舗のSEOキーワード戦略を立案してほしい。

【店舗情報】
- 業種：
- 店舗名：
- 所在地：高知県（市区町村まで）
- 強み・特徴：
- ターゲット客層：

【調査してほしいこと】
1. メインキーワード候補（月間検索数・競合強度つき）
   - 地名×業種系（例：高知市 美容室）
   - 悩み×解決系（例：高知 髪質改善）
   - 特徴×地名系（例：高知 カラー得意 美容室）

2. ロングテールキーワード候補10個以上
   （検索数は少ないが上位表示しやすいもの）

3. 競合サイトが上位を取っているキーワードと割って入るための戦略

4. 「今すぐ客」が使う検索ワードと「まだ迷っている客」が使う検索ワードの分類

【出力形式】
優先度A/B/C付きのキーワードリスト＋各キーワードの狙い方コメント
```

## プロンプト②：MEO最適化（Agent19 SEO Agent用）

```
以下の店舗のGoogleビジネスプロフィール（MEO）を最適化するための
具体的な改善指示書を作成してください。

【店舗情報】
- 業種：
- 店舗名：
- 所在地：
- 営業時間：
- 定休日：
- 強み・こだわり（3つ以上）：
- よく来る客層：
- メニュー・サービス名と価格帯：
- 現在のGBP状況：（口コミ数・評価・写真枚数・投稿頻度など）

【作成してほしいもの】
1. GBPビジネス説明文（750文字以内・キーワード自然に含む）
2. 投稿テンプレート3パターン（キャンペーン告知・季節ネタ・スタッフ紹介）
3. 口コミ返信テンプレート（高評価用・低評価用）
4. 写真・動画の撮影チェックリスト
5. 上位表示のために今すぐやるべき改善点リスト（優先度順）
```

## プロンプト③-a：コンテンツSEO ページ（Agent13 Copywriter用）

```
以下の条件で、SEOに強いWebページのコンテンツを作成してください。

【基本情報】
- 業種：
- 店舗名：
- 所在地：
- メインキーワード：
- サブキーワード（2〜3個）：
- ページの種類：

【作成してほしいもの】
1. titleタグ（32文字以内）
2. meta description（120文字以内）
3. h1見出し（1つ）
4. h2見出し構成案（5〜7個）
5. 各セクションの本文（h2ごとに200〜300文字）
6. FAQセクション（Q&A形式・5問）

【制約】
- キーワードの詰め込みはNG。自然な文章の中に含める
- 読み手は40〜60代の地方在住者。難しい言葉は使わない
- 「地元で長く愛されている」という文脈を大切にする
```

## プロンプト④：テクニカルSEO（Agent18 QA Agent用）

```
以下の条件のWebサイトに対して、テクニカルSEOの
チェックリストと改善指示書を作成してください。

【サイト情報】
- 技術スタック：Astro + Tailwind CSS
- ホスティング：Cloudflare Pages
- CMS：microCMS
- 業種：
- ページ数（想定）：

【チェックしてほしい項目】
1. Core Web Vitals対策（LCP・CLS・INP）
2. モバイル対応チェックリスト（40〜60代スマホユーザー想定）
3. 構造化データ（Schema.org）の実装指示
   - LocalBusinessスキーマの記述例
   - BreadcrumbListの記述例
   - FAQPageの記述例
4. Astro + Cloudflare Pages環境でのサイトマップ・robots.txt設定方法
5. 画像最適化チェックリスト
6. 公開前に必ず確認すべき20項目チェックリスト

【出力形式】
優先度（高・中・低）付きで、実装コード例も含めて出力してください。
```

## プロンプト⑤：月次レポート分析（Agent23・24用）

```
以下のWebサイトデータを分析して、
クライアント向けの月次SEOレポートを作成してください。

【基本情報】
- 業種：
- 店舗名：
- 計測期間：
- 前月比較期間：

【Google Search Consoleデータ】
- 表示回数：
- クリック数：
- 平均クリック率：
- 平均掲載順位：
- 上位クエリTOP10：

【Googleアナリティクスデータ】
- セッション数：
- ユーザー数：
- 直帰率：
- 平均セッション時間：
- 問い合わせ件数（コンバージョン）：

【GBPデータ】
- 検索表示回数：
- 地図表示回数：
- 電話タップ数：
- ルート検索数：
- 写真閲覧数：

【作成してほしいもの】
1. 今月のサマリー（3行・オーナーが一目でわかる言葉で）
2. 前月比の評価（良かった点・課題点）
3. 来月取り組むべき改善アクション（優先度順・3〜5個）
4. オーナーへの説明文（専門用語なし・口語調）

【制約】
- 数字の羅列ではなく「何が起きているか」を日本語で説明する
- ネガティブな数字も「次の打ち手」とセットで伝える
- A4 1枚に収まるボリューム感で
```

## 実務での使い方フロー

```
契約後・制作前
  └ ① Agent19.5 キーワード戦略
      ↓ Task DBにキーワードリストを保存
制作中
  └ ③-a Agent13 Copywriter
      ↓ キーワードを自然に含む文章を生成
公開前
  └ ④ Agent18 QA テクニカルSEOチェック
      ↓ 問題なければ公開
公開直後
  └ ② Agent19 SEO → MEO最適化指示書を生成
      ↓ GBPを整備
毎月
  └ ⑤ Agent23・24 月次レポート分析
      ↓ 改善 → ③-b Agent25 ブログ記事提案
```
